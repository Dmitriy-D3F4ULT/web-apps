Введение в JavaScript

Давайте посмотрим, что такого особенного в JavaScript, чего можно достичь с его помощью и какие другие технологии хорошо с ним работают.

Что такое JavaScript?

Изначально JavaScript был создан, чтобы «сделать веб-страницы живыми».

Программы на этом языке называются скриптами. Они могут встраиваться в HTML и выполняться автоматически при загрузке веб-страницы.

Скрипты распространяются и выполняются, как простой текст. Им не нужна специальная подготовка или компиляция для запуска.

Сегодня JavaScript может выполняться не только в браузере, но и на сервере или на любом другом устройстве, которое имеет специальную программу, называющуюся «движком» JavaScript.

Что может JavaScript в браузере?

Современный JavaScript – это «безопасный» язык программирования. Он не предоставляет низкоуровневый доступ к памяти или процессору, потому что изначально был создан для браузеров, не требующих этого.

Возможности JavaScript сильно зависят от окружения, в котором он работает. Например, Node.js поддерживает функции чтения/записи произвольных файлов, выполнения сетевых запросов и т.д.

В браузере для JavaScript доступно всё, что связано с манипулированием веб-страницами, взаимодействием с пользователем и веб-сервером.

Например, в браузере JavaScript может:

  1. Добавлять новый HTML-код на страницу, изменять существующее содержимое, модифицировать стили.
  2. Реагировать на действия пользователя, щелчки мыши, перемещения указателя, нажатия клавиш.
  3. Отправлять сетевые запросы на удалённые сервера, скачивать и загружать файлы (технологии AJAX и COMET).
  4. Получать и устанавливать куки, задавать вопросы посетителю, показывать сообщения.
  5. Запоминать данные на стороне клиента («local storage»).

Что делает JavaScript особенным?

Как минимум, три сильные стороны JavaScript:

  1. Полная интеграция с HTML/CSS.
  2. Простые вещи делаются просто.
  3. Поддерживается всеми основными браузерами и включён по умолчанию.

JavaScript – это единственная браузерная технология, сочетающая в себе все эти три вещи.

Вот что делает JavaScript особенным. Вот почему это самый распространённый инструмент для создания интерфейсов в браузере.

Хотя, конечно, JavaScript позволяет делать приложения не только в браузерах, но и на сервере, на мобильных устройствах и т.п.

Итого

  1. JavaScript изначально создавался только для браузера, но сейчас используется на многих других платформах.
  2. Сегодня JavaScript занимает уникальную позицию в качестве самого распространённого языка для браузера, обладающего полной интеграцией с HTML/CSS.
  3. Многие языки могут быть «транспилированы» в JavaScript для предоставления дополнительных функций. Рекомендуется хотя бы кратко рассмотреть их после освоения JavaScript.

Пример №3:

Запросить у пользователя любое число больше 5, в случае правильного ответа выдать окошко с поздравлением, случае неправильного ответа вывести повторное окно с вопросом (помните, что пользователь может нажать "отмена", помните о типах данных)

Решение:

Здесь мы используем новую функцию под названием "while" - (это цикл) с условием "true", то есть он будет работать, пока не сработает команда "break".
(Существует 3 вида циклов "for", "while", "dowhile")
В нашем цикле мы избавляемся от всех неправильных вариантов ввода путём условных ветвлений "if/else".

В первом сравнении мы проверяем, а является ли ввод числом и только им, с помощью команды "isNaN" - (Not a Number), если результат сравнение "true", значит вменяемое число мы не получили.
С помощью "continue" мы сразу переходим на следующую итерацию, не проверяя остальные варианты.

Возможно вы уже догадались как упростить код до пары строк, так зачем нам столько сравнений?.
Всё просто! Для интересной вариативности запросов и лучшего понимания условных ветвлений :)

Продолжим. Во втором сравнении мы реагируем на то, если пользователь нажмёт "Отмена".

В третьем, оповещаем, что введённое число меньше или равно 5.
Кстати, тут и после мы используем логический оператор "&&", который означет "И". Также из простых операторов существуют такие как: ("||"-ИЛИ) и ("!"-НЕ).
В этом сравнении нам нужно, чтобы isNaN был равен false, то есть ввод являлся числом И чтобы это число (наша переменная num) было меньше 5 (num < 5).
Зачем перед num написано "Number"? Потому что весь ввод передаётся в код как строка (даже если он состоит только из цифр), и таким образом мы изменяем тип нашей num из "string" в тип "number" для дальнейшего использования в роли числа.

И наконец, если все условия соблюдены, мы поздравляем пользователя и завершаем работу цикла.

  1. let num = prompt("Введите любое число больше 5");
  2. while (true) {
  3.     if (isNaN(num) == true) {
  4.         num = prompt("ВВЕДИТЕ ЛЮБОЕ ЧИСЛО БОЛЬШЕ 5!");
  5.         continue;
  6.     } else if (typeof(num) == "object") {
  7.         num = prompt("Никаких отмен, введите любое число больше 5");
  8.         continue;
  9.     } else if (isNaN(num) == false && Number(num) < 6) {
  10.         num = prompt("Это меньше или равно 5, введите любое число больше 5");
  11.         continue;
  12.     } else if (isNaN(num) == false && Number(num) > 5) {
  13.         alert("Верно! Поздравляю!");
  14.         break;
  15.     }
  16. }

Пример №4:

Выведите нечетные числа от 1 до 7, пропустив 5

Решение:

Здесь мы используем новый цикл "for". Но что же в его скобках?

for (начало; условие; шаг)
Разберёмся подробнее на нашем примере:
(let i = 1) - задаёт переменную и её величину, от которой пойдут дальнейшие итерации. Так как нам нужно выводить числа от единицы, единицу и присвоим.
(i < 8) - условие того, что цикл будет работать, пока i не достигнет значения 8, то есть когда это условие сработает, цикл завершится не исполнив код внутри него.
(i += 2) - шаг, (вариант более короткой записи "i = i + 2") то есть сколько будет прибавляться к i после каждой итерации. Так как наше начало единица и нам нужно выводить нечётные числа, мы прибавляем 2.

А дальше всё просто, по заданию нам нужно пропустить пятёрку, создаём условие, если i равно 5, то переходим на следующую итерацию. В остальных случаях выводим i.

  1. for (let i = 1; i < 8; i += 2) {
  2.     if (i == 5) {
  3.         continue;
  4.     } else {
  5.         alert(i);
  6.     }
  7. }

Пример №5:

Выведите все элементы массива, используя цикл for

Решение:

Здесь мы впервые сталкиваемся с массивами. В данном случае с самыми простыми. Массивы создаются с помощью квадратных скобок [].
Назовём это хранилищем элементов, где к каждому из них мы можем обратиться и использовать в своих целях. Массив может содержать элементы разных типов, его можно уменьшать, увеличивать или заменять в нём элементы.

И так, у нас есть массив из чисел [1, 5, 8, 2, 4, 6], который последовательно нужно вывести на экран.

В цикле мы используем условие "пока i меньше длины массива, она равна кол-ву в нём элементов", а также шаг "i++", который означает i+=1.

Чтобы выводить элементы массива нам потребуется обратиться к их индексам, у каждого элемента есть свой индекс. Но индексация в массиве начинается с "0". Поэтому начало нашего цикла равняется нулю. И условие будет работать, так как размер нашего массива 6, но индекс последнего элемента будет равен 5.

  1. let arr = [1, 5, 8, 2, 4, 6];
  2. for (let i = 0; i < arr.length; i++) {
  3.     alert(arr[i]);
  4. }

Пример №6:

Создайте 2 массива. Объедините их в один и выведите на экран полученный массив.
Удалите последний член получившегося массива и выведите это значение на экран.
Добавьте новый элемент в начало вашего объединенного массива.

Решение:

Для наглядности работы кода, будет много выводов на экран.
Создаём два числовых массива. И в цикле for, с помощью команды ".push" добавляем в первый массив элементы из второго обычным перебором.

Далее нам нужно удалить последний элемент, делаем это с помощью команды ".pop". Если выводить её на экран, то она покажет удаляемый элемент, а не уже изменённый массив.

Ну а теперь нужно добавить элемент в начало нашего массива с помощью команды ".unshift". Как вы видите, мы добавляем элемент "0". А затем выводим наш переработанный массив.

  1. let arr1 = [1, 2, 3, 4], arr2 = [5, 6, 7, 8];
  2. for (let i = 0; i < arr2.length; i++) {
  3.     arr1.push(arr2[i]);
  4. }
  5. alert(arr1);
  6. alert(arr1.pop());
  7. alert(arr1);
  8. arr1.unshift(0);
  9. alert(arr1);

Пример №7:

Выведите сегодняшнюю дату в формате 11 ноября 2020 года

Решение:

Здесь "new Date" выдаёт нам сегодняшнюю дату, но не в нашем формате, а с помощью ".toLocaleDateString" мы преобразуем её под наш личный регион, однако дата выдаётся нам в виде строки и числовым форматом (дд.мм.гггг)("22.11.2024").

Чтобы понять какой месяц нам вывести на экран, мы делим нашу строку (делителем высутупают точки) на элементы и записываем их в массив. Таким образом получится ["22", "11", "2024"].

С помощью switch мы сравниваем 2 элемент в массиве с индексом "1", то есть числовой месяц с их названиями. А затем присваиваем переменной "month" название нашего месяца.

В заключении выводим нужный нам формат даты используя обращение к элементам массива, нашу переменную "month" и дописав слово "года".

  1. let now = new Date().toLocaleDateString();
  2. let date = now.split(".");
  3. let month;
  4. switch (date[1]) {
  5.     case "01": month = "января"; break;
  6.     case "02": month = "февраля"; break;
  7.     case "03": month = "марта"; break;
  8.     case "04": month = "апреля"; break;
  9.     case "05": month = "мая"; break;
  10.     case "06": month = "июня"; break;
  11.     case "07": month = "июля"; break;
  12.     case "08": month = "августа"; break;
  13.     case "09": month = "сентября"; break;
  14.     case "10": month = "октября"; break;
  15.     case "11": month = "ноября"; break;
  16.     case "12": month = "декабря"; break;
  17. }
  18. alert(`${date[0]} ${month} ${date[2]} года`);
